# Day 01 – NIFTY 50 Market Data Analysis (Excel)

## Dataset Overview
Analyzed historical NIFTY 50 index data containing:
- Date
- Open Price
- High Price
- Low Price
- Close Price
- Shares Traded
- Turnover (₹ Cr)

## Objective
To analyze market trends and understand price movement patterns using Microsoft Excel.

## Tasks Performed
- Cleaned and structured raw financial data
- Analyzed daily price movement (Open vs Close)
- Examined volatility using High and Low values
- Reviewed trading activity through Shares Traded and Turnover
- Created a line chart to visualize closing price trends
